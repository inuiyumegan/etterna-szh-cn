# ENGLISH

**Two versions are provided. Please choose the one that suits your needs:**

1. Old fonts version – Some characters may display incorrectly and it cannot be used with the Til Death theme.
2. New fonts version – No known font issues so far and compatible with the Til Death theme.

- For the old fonts version, the reason it cannot work with the Til Death theme is that the Rebirth theme has relatively good support for Chinese fonts, while Til Death theme does not support them properly.
Therefore, players who prefer the Til Death theme should install the new font version.

* Some characters cannot be displayed in the old font set, such as “运适关响极种”.
  Because of this limitation, the Help interface is not translated in the old font version.
  If you want to view the Etterna Help interface in Simplified Chinese, please install the new font version.

**How to use:**

* It is recommended to back up your Etterna installation before use.

* Choose one version and copy it into the Etterna folder, replacing all files.

* After launching the game, change the language to “szh” and restart the game.
  *(Restarting is very important, otherwise a lot of garbled text may appear.)*

* *Currently, the new font version is recommended, as the font loading speed has been greatly improved.*

## Common Issues

* Installing the Simplified Chinese translation will not overwrite Traditional Chinese, but the Fonts folder will be updated.

* If you want to switch between different font versions, it is recommended to download two separate Etterna installations, or manually replace the Fonts folders in:

```
  Etterna\Themes\Til Death
  Etterna\Themes\Rebirth
  Etterna\Themes\_fallback
```

* If errors occur after installing the translation multiple times, try deleting the Fonts and BGAnimations folders in:

```
  Etterna\Themes\Til Death
  Etterna\Themes\Rebirth
  Etterna\Themes\_fallback
```

  Then reinstall the latest version of the translation.

# 中文

**分为两个版本，请按需使用：**
1. 旧字库版，字体显示有问题，且无法适配Til Death主题；
2. 新字库版，字体目前暂无发现问题，可适配Til Death主题。

- 对于旧字库版，无法适配Til Death主题的原因是，Rebirth主题对中文字库支持比较好，Til Death则完全没法用，
所以对于习惯Til Death主题的玩家，请安装新字库版。

- 旧字库有部分字无法使用，如“运适关响极种”，所以旧字库版的帮助界面不做翻译，
如想查看Etterna帮助界面简中版，请安装新字库版。


**使用方式：**

- 建议使用前先备份一遍Etterna。
- 选择一个版本复制到Etterna文件夹并覆盖所有文件。
- 打开游戏后，将语言切换为“szh”并重启游戏（重启非常重要，否则会出现很多乱码）即可。

- *目前推荐安装新字库版，字库加载速度已得到极大提升。*

**常见问题：**

- 安装简中不会覆盖繁中，但字库（Fonts）会更新。

- 若想使用不同版本的字库，建议下载两个Etterna，或在游玩前替换
```
  Etterna\Themes\Til Death
  Etterna\Themes\Rebirth
  Etterna\Themes\_fallback
```
中的Fonts文件夹。

- 如果重复安装汉化后报错，可以尝试删除
```
  Etterna\Themes\Til Death
  Etterna\Themes\Rebirth
  Etterna\Themes\_fallback
```
中的Fonts和BGAnimations文件夹，然后再安装最新版汉化。
